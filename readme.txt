Enables Notifyfox on all user available pages.

== Description ==

This plugin adds the required javascript for notifyfox notifications. And all the users visiting your website will be prompted with a confirmatory box for allowing your website to send them push notifications

For more information visit:

[Notifyfox Notifications](https://notifyfox.com)

<h4>Checkout this video tutorial</h4>
[How Notifyfox works](https://www.youtube.com/watch?v=q9KitWvMsQI&t)

= Features =

* Notify your website subscribers whenever you have a new post, events anything.
* Can add images in the notification along with Message and a landing url.
* Supports Multiple authors.
* Author can choose to send notifications to all the subscribers or subscribers of that particular author.
* Simple and easy to use
* WPML Compatible.
* Non intrusive styles, looks perfect on all the devices. Supports Android, Mac , Linux and Ubuntu.

== Installation ==

There's 3 ways to install this plugin:

= 1. The super easy way =
a. In your Admin, go to menu Plugins > Add New
b. Search for NotifyFox
c. Click to install
d. Activate the plugin
e. A new menu `NotifyFox` will appear in your Admin
f. Add the User ID, User Name, Client ID and Client Secret from Notifyfox to the settings (Admin > Notifyfox > Settings > Advanced Setting)

= 2. The easy way =
a. Download the plugin (.zip file) on the right column of this page
b. In your Admin, go to menu Plugins > Add New
c. Select the tab "Upload"
d. Upload the .zip file you just downloaded
e. Activate the plugin
f. A new menu `NotifyFox` will appear in your Admin
g. Add the User ID, User Name, Client ID and Client Secret from Notifyfox to the settings (Admin > Notifyfox > Settings > Advanced Setting)

= 3. The old way (FTP) =
a. Upload `notifyfox-plugin-zip` folder to the `/wp-content/plugins/` directory
b. Activate the plugin through the 'Plugins' menu in WordPress
c. A new menu `NotifyFox` will appear in your Admin
d. Add the User ID, User Name, Client ID and Client Secret from Notifyfox to the settings (Admin > Notifyfox > Settings > Advanced Setting)
